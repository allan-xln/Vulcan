# Roadmap By Phase

## Phase 1
- Establish monorepo, conventions, local infrastructure and starter services.

## Phase 2
- Implement Supabase schema, tenant model, RLS policies, audit tables and seeded local fixtures.

## Phase 3
- Build ingestion gateway with signed agent registration, event validation and ClickHouse writes.

## Phase 4
- Add control-plane web UI for tenant admin, data source visibility and audit exploration.

## Phase 5
- Add explainable scoring pipeline with traceable formulas and evidence references.

## Phase 6
- Add alerting, rate limits, backpressure controls and operational dashboards.

## Phase 7
- Add CI, release controls, migration discipline and production hardening.
